# EISMaster

EISMaster is a Windows desktop tool for parsing CHI660F EIS data, exporting TXT tables for plotting/DRT/fitting workflows, fitting a small set of lithium-ion equivalent circuits, and tracking parameter trends for operando batches.

## Run

```powershell
C:\Users\chs\.conda\envs\py311\python.exe -m pip install -e .[advanced]
C:\Users\chs\.conda\envs\py311\python.exe -m eismaster
```

## Supported inputs

- `.bin` generated by CHI `A.C. Impedance` experiments on CHI660F-class files validated against the sample in this workspace
- `.txt` exports from CH Instruments software
- `.csv`/delimited exports with `frequency, Zreal, Zimag` style columns

## TXT exports

- `plot_data.txt`
- `drt_input.txt`
- `fit_input.txt`
- `fit_params.txt`
- `fit_report.txt`
- `batch_summary.txt`
